Copy the code below! This is the code + pinout to make the circuit python serial print out RH and temp every 2 seconds. On the circuit board: Trinket M0 Pin 0 --> SHT31 SDA Trinket M0 Pin 2 --> SHT31 SCL Trinket M0 Gnd --> SHT31 GND Trinket M0 3V --> SHT31 VIN import board import busio import adafruit_sht31d import time i2c = busio.I2C(board.SCL, board.SDA) sensor = adafruit_sht31d.SHT31D(i2c) while True: print("Temp:", sensor.temperature) print("Humidity:", sensor.relative_humidity) print("-----") time.sleep(2)
